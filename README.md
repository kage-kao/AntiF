*AntiF* - продвинутый Anti-Detect браузер для Android.

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🎭 **Multi-Profile** | Неограниченные изолированные профили браузера |
| 🔐 **Fingerprint Spoofing** | Canvas, WebGL, Audio, Screen, Timezone |
| 🚫 **AdBlock Engine** | 250+ заблокированных доменов + URL паттерны |
| 🌐 **Proxy Support** | HTTP и SOCKS5 с авторизацией |
| 📑 **Multi-Tab** | Несколько вкладок в каждом профиле |
| 🍪 **Cookie Manager** | Просмотр, добавление, удаление cookies |
| 💻 **JS Console** | Выполнение JavaScript команд |

---

## 🎭 Что спуфится

```
✅ navigator.userAgent      ✅ Canvas fingerprint
✅ navigator.platform       ✅ WebGL vendor/renderer  
✅ Screen resolution        ✅ AudioContext
✅ Hardware concurrency     ✅ Timezone
✅ Device memory            ✅ Client Rects
✅ WebRTC (blocked)         ✅ Battery API
✅ FingerprintJS (blocked)  ✅ Font detection
```

- **AndroidX WebKit** — продвинутый WebView
- **Material Design 3** — тёмная тема
