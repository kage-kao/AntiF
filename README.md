# AntiF Browser

Android браузер с обходом ограничений и защитой приватности.

## Возможности

### Автоматический Bypass (включены по умолчанию)
- **FULL BYPASS** — полный обход ограничений
- **Block Tracking** — блокировка трекеров (FingerprintJS, PostHog, Facebook, TikTok)
- **Spoof Feature Flags** — подмена флагов функций
- **New Device-Id** — генерация нового ID устройства каждую сессию
- **Kill PostHog** — нейтрализация аналитики PostHog
- **Turkey / Canada (Free)** — эмуляция региона для бесплатного доступа

### Защита приватности
- Спуфинг fingerprint браузера
- Изоляция cookies между профилями
- Подмена данных в запросах к аналитике (ap.emergent.sh)
- AdBlock с косметическими фильтрами

### Дополнительно
- Мульти-профили с изолированными данными
- Консоль для выполнения JavaScript команд
- Система логирования для отладки
- Генератор тестовых карт

## Использование

### Главный экран
- **⚡ Bypass** — настройки обхода (галочки)
- **📋 Logs** — просмотр/сохранение логов
- **🗑️** — очистка всех данных

### В браузере
- **Консоль (>_)** — выполнение JS команд
- **Меню (⋮)** — быстрые действия, AdBlock, приватность

## Как работает

1. **Блокировка** — FingerprintJS, PostHog, Facebook Pixel полностью блокируются
2. **Спуфинг** — запросы к ap.emergent.sh проходят, но с фейковым device-id
3. **Результат** — сайт работает, но не может связать тебя с предыдущими сессиями

## Лицензия

Только в образовательных целых.
